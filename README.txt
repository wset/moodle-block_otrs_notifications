
OTRS Notifications Block
========================

An extension to the OTRS Help Desk Block to create ticket notifications on quiz
attempts and course completions.

Setup
=====

Install & Setup the OTRS Help Desk Block.

Place block files in /block/otrs_notifications within the moodle tree and install 
plugin in the usual way.

Setup queues any any dynamic fields desired for the grades, marks, etc in OTRS and
update the settings in the OTRS Help Desk Block accordingly.

Add block to course.  Within the instance settings activate course completion
notifications and/or select quizes to send attempt notifications for.
